<h1>HOW TO USE</h1>

1 - enter in the first column of a Google Sheet the list of IDs of the container to be processed, like this:

<b>Container ID</b><br />
GTM-XXXXXXX<br />
GTM-YYYYYYY<br />
GTM-WWWWWWW<br />
GTM-ZZZZZZZ<br />

2 - replace the string SPREADSHEET_ID in the script with the ID of the Google Sheet.
